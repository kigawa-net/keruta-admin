# keruta-admin

keruta管理パネルは、タスク・ドキュメント・リポジトリ・Kubernetesリソースを管理するWebインターフェースです。

## 技術スタック
- **フレームワーク**: Remix.js (React)
- **スタイリング**: Bootstrap 5
- **言語**: TypeScript

## 開発方法
### 環境構築
1. 依存関係のインストール
```bash
npm install
```

2. 開発サーバーの起動
```bash
npm run dev
```

3. ブラウザで http://localhost:3000 にアクセス
